The shittest module on NPM.

Feces is an NPM module that returns photos of shit from r/poop.

CLI

Just run npx feces.

Library

Feces works with both ESM Modules and CommonJS. We aren't shitty.

In Node... 
npm i feces
In the browser... 
const fucking_shit = await import('https://cdn.pika.dev/feces@2').then(module=>module.default)

The library is just a function that returns a promise that resolves to the URL.

  import feces from 'feces'
  feces().then(console.log)
