The shittest module on NPM.

Feces is an NPM module that returns photos of shit from r/poop.

CLI

Just run npx feces.

Library

Feces works with both ESM Modules and CommonJS. We aren't shitty.

The function returns a promise that resolves to the URL.

import feces from 'https://cdn.pika.dev/feces'
feces().then(console.log) // log shit to the console
